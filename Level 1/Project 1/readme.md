# Retail Sales Exploratory Data Analysis

## Objective
To perform Exploratory Data Analysis (EDA) on retail sales data to uncover trends, patterns, and insights that can support business decision-making.

## Dataset
Retail Sales Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
- Data loading and cleaning
- Handling missing values
- Descriptive statistics
- Time-based sales analysis
- Customer and product analysis
- Data visualization

## Key Insights
- Seasonal trends impact total sales
- Certain product categories generate higher revenue
- Customer demographics influence purchasing behavior

## Outcome
Developed strong skills in data cleaning, EDA, and visualization techniques.
