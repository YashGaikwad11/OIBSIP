# Twitter Sentiment Analysis

## Objective
To analyze public sentiment from Twitter data using text analytics techniques.

## Dataset
Twitter Sentiment Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- NLP techniques

## Steps Performed
- Text cleaning and preprocessing
- Sentiment classification
- Visualization of sentiment distribution

## Key Insights
- Majority of tweets fall into neutral and positive categories
- Sentiment analysis helps understand public opinion trends

## Outcome
Improved understanding of text preprocessing and sentiment analytics.
