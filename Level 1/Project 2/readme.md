# Customer Segmentation Analysis

## Objective
To segment customers based on demographic and purchasing behavior using data analytics techniques.

## Dataset
Marketing / Customer Segmentation dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Steps Performed
- Data preprocessing
- Feature selection
- Customer behavior analysis
- K-Means clustering
- Cluster visualization

## Key Insights
- Customers can be grouped into distinct segments
- High-income customers show different spending patterns
- Segmentation helps in targeted marketing strategies

## Outcome
Gained practical experience in clustering and customer analytics.
