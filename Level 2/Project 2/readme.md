# Google Play Store Data Analysis

## Objective
To analyze Google Play Store data to understand app market dynamics, trends, and user preferences.

## Dataset
Google Play Store Apps Dataset from Kaggle.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
- Data cleaning and type correction
- Category-wise app analysis
- Rating, installs, and pricing analysis
- Sentiment estimation based on ratings
- Data visualization

## Key Insights
- Most apps on Play Store are free
- Higher ratings correlate with higher installs
- Games and communication apps dominate the market

## Outcome
Enhanced skills in real-world data cleaning and market analysis.
