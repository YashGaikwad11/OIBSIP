Support Vector Classifier
# Techniques Used
Data scaling
Correlation analysis
Confusion matrix
Accuracy comparison
# Libraries
Pandas
NumPy
Seaborn
Matplotlib
Scikit-learn
# Output
The project compares different classifiers and identifies the best-performing model.
